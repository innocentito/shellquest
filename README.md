Interactive terminal game for mastering bash/zsh commands through gamified learning.

## Features

- **XP & Leveling**: Level up with streak bonuses (up to 3x multiplier)
- **15+ Achievements**: Unlock Common to Legendary achievements
- **5 Question Types**: Multiple choice, fill-in-blank, fix errors, command builder, what-does-it-do
- **Story Mode**: Learn through narrative chapters
- **Mystery Cases**: Solve command-line investigations
- **Battle Mode**: Compete with friends over LAN
- **Progress Tracking**: Auto-save with detailed statistics
- **Smart Difficulty**: Questions adapt to your weaknesses

## Installation

```bash
git clone https://github.com/innocentito/shellquest.git
cd shellquest
./install.sh
```

## Usage

```bash
shellquest
# or: python3 -m shellquest
```

## Game Modes

- **Quick Quiz**: Practice essential or advanced commands
- **Story Mode**: Progress through narrative chapters
- **Mystery Cases**: Solve investigations using shell commands
- **Battle Mode**: Real-time 1v1 over LAN
- **Command Training**: Focus on specific commands

## Controls

- `H` - Show hint (-50% XP)
- `S` - Skip question
- `Q` - Return to menu

## Requirements

- Python 3.8+
- Terminal with color support
